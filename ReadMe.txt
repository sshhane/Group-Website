INTRODUCTION
------------

This programme takes in either a text file or url and parses the contents into an arrayList.

There is functionality to perform a variety of searches on the arrayList, and an option to display
statistics of the arrayList.

INSTRUCTIONS
------------

Select options using the relevant number
e.g.	1) Parse a File or URL
	2) Search
	3) Print Stats
Enter '1' for Parse a File or URL etc.
